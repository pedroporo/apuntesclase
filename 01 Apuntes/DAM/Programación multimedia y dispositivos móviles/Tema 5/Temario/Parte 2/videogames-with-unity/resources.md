## Unity Scripting API

The Unity Scripting API offers a robust suite of tools for developers to control the Unity engine through code. It provides numerous classes, methods, and properties for creating custom behaviors, manipulating game objects, and handling input, making it essential for game development.

[![Logo](https://ecm-pmdm-unity.gitbook.io/videogames-with-unity/~gitbook/image?url=https%3A%2F%2Fdocs.unity3d.com%2F2022.3%2FDocumentation%2FStaticFiles%2Fimages%2Ffavicons%2Fapple-touch-icon-152x152.png&width=20&dpr=4&quality=100&sign=ec8ee864&sv=2)Unity - Scripting API:docs.unity3d.com](https://docs.unity3d.com/2022.3/Documentation/ScriptReference/index.html)

## Unity Asset Store

The Unity Asset Store is a marketplace for game developers to find, buy, and sell assets and tools. It offers 3D models, animations, scripts, plugins, and more to streamline game development and enhance project quality.

[![Logo](https://ecm-pmdm-unity.gitbook.io/videogames-with-unity/~gitbook/image?url=https%3A%2F%2Fassetstore.unity.com%2Ffavicon.ico&width=20&dpr=4&quality=100&sign=1fd3d82&sv=2)Unity Asset Storeassetstore.unity.com](https://assetstore.unity.com)
