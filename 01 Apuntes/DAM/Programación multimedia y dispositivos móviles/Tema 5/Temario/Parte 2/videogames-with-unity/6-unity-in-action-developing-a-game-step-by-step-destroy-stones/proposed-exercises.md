# Proposed Exercises

Source: https://ecm-pmdm-unity.gitbook.io/videogames-with-unity/6.-unity-in-action.-developing-a-game-step-by-step-destroy-stones/proposed-exercises

To enhance "DestroyStones" and make it a great gaming experience, consider incorporating the following upgrades:

**1. Enhanced User Interface:**

* **Initial Screen:** Introduce an appealing startup screen with game title and play instructions.
* **Configuration Screen:** Provide settings options for sound, difficulty levels, and controls customization.
* **Final Screen:** Design an engaging game over screen showcasing scores and replay option.
* **Enhance Scene Background:** Improve the visual appeal of your scenes, for example:

  + Change the **camera** **background** **color**, or
  + Add a **UI** **Image to the Canvas** (Ensure that the imported image Texture Type is set to "Sprite" and verify that the UI Image is the first element within the Canvas hierarchy to appear behind all other UI elements.), or
  + Change the default **Camera** **Skybox** to a new one you can find in Asset store and import it to project. Then, go to menu *Window -> Rendering -> Lighting -> Environment,*and assign your new skybox material to the *Skybox Material* field.

**2. Sound Effects:**

* Use engaging **background** **music** to improve the game atmosphere.
* Integrate **explosion sounds** when stones are destroyed to enhance the action feel.

**3. Progressive Gameplay:**

Introduce a progressive series of **levels** with increasing difficulty. As players advance, the **speed** increases and **challenges** are more demanding.

**4. Gameplay Elements:**

* Add **new** **targets**, referred to as **"Friends,"**  that players must **avoid** **destroying**. Penalties apply for mistakes.

**5. Other**

* Incorporate a variety of **lighting** options, diverse **font styles**, ...., etc.

Implementing these features will transform "DestroyStones" into a more immersive and challenging game for a wider audience.
